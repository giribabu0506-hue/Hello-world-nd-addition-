AuraGB Programming Language

AuraGB is a customized programming language designed for simple, human-readable syntax and lightweight execution. It is specially focused on running in low-resource environments such as keypad mobile phones and embedded systems.

---

👨‍💻 Author

AuraGB was created and developed by Giri Babu
GitHub: https://github.com/giribabu0506

---

🚀 Features

- Very simple and readable syntax
- Beginner-friendly commands
- Lightweight execution model
- Suitable for embedded systems and keypad devices
- Fast and minimal design

---

📌 Example Program

Code

add 2 4

Output

6

---

🧠 Basic Syntax

Command| Description| Example
add| Adds two numbers| add 2 4
set| Assign value (future scope)| set a 5
